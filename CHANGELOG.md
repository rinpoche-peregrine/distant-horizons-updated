# Changelog

This is a fork of [Distant Horizons](https://gitlab.com/distant-horizons-team/distant-horizons) by the original team.

## 3.0.0-b-dev (MC 26.1)

- Updated Distant Horizons for Minecraft 26.1
- Changed default rendering API from BLAZE_3D to OPEN_GL for Iris shader compatibility
- Removed bundled Fabric API to prevent version conflicts with modpacks
- Removed stale google-collect dependency
